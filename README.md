# TODO_LIST(API)

## with Backend: (Laravel & MySQL)
## with Frontend: (React)
### For Install and run Project:
#### - git clone https://github.com/raneenmahmoud/TODO_LIST.git

### For Front-React:
#### - go to todo-list-app
#### - run npm install
#### - run npm start

### For Backend-Laravel:
#### - go to todo-list-api
#### - run composer install
#### - cp .env.example .env 
#### - name your database ia todo_list
#### - php artisan key:generate
#### - php artisan migrate
#### - php artisan db:seed
#### - php artisan serve
